How to Play

Objective

Fill the 9×9 grid so that each column, each row, and each of the nine 3×3 sub-grids contains all digits from 1 to 9.

How to Play

  • Click on any empty cell to select it
  • Type a number (1-9) or click the number pad
  • Use backspace or the eraser to clear a cell
  • The game validates your input in real-time

Features

  • Auto Solve: Watch the AI solve the puzzle step-by-step
  • Hints: Get help when you're stuck
  • Validation: Check if your current progress is correct
  • Multiple Difficulties: From Easy to Expert levels

Keyboard Shortcuts

  • 1-9: Enter numbers
  • Backspace or Delete: Clear cell
  • Arrow Keys: Navigate between cells
  • Space: Get hint
  • Escape: Deselect cell
